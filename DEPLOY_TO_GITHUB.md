# 🚀 Deploy Bitesu India Website to GitHub Pages

## What you'll get
- Free hosting at `https://yourusername.github.io/bitesuindia`
- Then point `bitesuindia.com` to it (free)

---

## Step 1 — Create a GitHub Account
Go to https://github.com and sign up (free). Use any email.

---

## Step 2 — Create a New Repository

1. Click the **"+"** button (top right) → **"New repository"**
2. Repository name: `bitesuindia` (or `bitesuindia.com`)
3. Set to **Public**
4. Tick **"Add a README file"**
5. Click **"Create repository"**

---

## Step 3 — Upload Your Files

1. Inside your new repo, click **"Add file"** → **"Upload files"**
2. Drag and drop **`index.html`** from your Lead generation folder
3. Click **"Commit changes"**

---

## Step 4 — Enable GitHub Pages

1. Go to your repo → **Settings** tab
2. Scroll down to **"Pages"** (left sidebar)
3. Under **"Branch"** → select **`main`** → folder: **`/ (root)`**
4. Click **Save**
5. Wait 2 minutes → your site is live at:
   `https://yourusername.github.io/bitesuindia`

---

## Step 5 — Point bitesuindia.com to GitHub Pages

### In GitHub:
1. Go to repo → **Settings → Pages**
2. Under "Custom domain" → type `www.bitesuindia.com` → Save

### In your Domain Registrar (GoDaddy / Namecheap / etc.):
Add these DNS records:

| Type  | Host | Value                  |
|-------|------|------------------------|
| A     | @    | 185.199.108.153        |
| A     | @    | 185.199.109.153        |
| A     | @    | 185.199.110.153        |
| A     | @    | 185.199.111.153        |
| CNAME | www  | yourusername.github.io |

Wait 10–30 minutes for DNS to update.

### Enable HTTPS:
Back in GitHub → Settings → Pages → tick **"Enforce HTTPS"**

---

## Step 6 — Add a CNAME file (important!)

In your GitHub repo, create a new file called `CNAME` (no extension) with just this inside:
```
www.bitesuindia.com
```

---

## ✅ Done!

Your new premium site will be live at **bitesuindia.com** within 30 minutes.

---

## Updating the site later

1. Edit `index.html` on your computer
2. Go to GitHub repo → click `index.html` → click pencil icon to edit
3. Or drag & drop the new file (it will replace the old one)
4. Changes go live in ~1 minute
